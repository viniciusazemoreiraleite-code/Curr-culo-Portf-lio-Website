# 🚀 Portfolio Pessoal | Vinícius Azeredo

Este é o repositório do meu portfolio profissional, desenvolvido para apresentar a minha trajetória, competências e projetos como **UI/UX e Web Designer**. O projeto foi construído focando-se em minimalismo, fluidez de navegação e uma experiência de utilizador otimizada.

---

## 🎨 O Design

O conceito visual foi criado para refletir uma estética limpa e profissional, utilizando uma paleta de cores terrosas e tipografia moderna.

- **Design System:** Baseado em tons de verde musgo (`#3D5A3D`), creme e bege, garantindo um alto contraste e legibilidade.
- **Tipografia:** Utilização da fonte **Poppins** para uma leitura clara e contemporânea.
- **Grid:** Layout estruturado com base em **1440px** para desktops, garantindo uma hierarquia visual equilibrada.

## 🛠️ Tecnologias e Ferramentas

Para garantir performance e facilidade de manutenção, utilizei:

- **HTML5 Semântico:** Para melhor acessibilidade e SEO.
- **CSS3 Personalizado:** Uso de **CSS Variables**, **Flexbox** e **CSS Grid** para um layout dinâmico.
- **JavaScript (Vanilla):** Implementação de lógica para troca de abas (Tabs), scroll suave e manipulação do header ao fazer scroll.
- **Vercel:** Plataforma utilizada para o deploy e CI/CD.

## 📱 Responsividade

O projeto foi totalmente otimizado para dispositivos móveis através de um fluxo **Desktop-First** com ajustes críticos em **768px**:

- **Header Adaptativo:** Menu reorganizado para melhor usabilidade em touchscreens.
- **Hero Section:** Empilhamento de elementos para manter o impacto visual no mobile.
- **Grid de Projetos:** Cards que se ajustam automaticamente para evitar cortes e garantir centralização.
